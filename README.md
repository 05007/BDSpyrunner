# 接口说明
## 监听器
设置监听器,执行函数<br>
执行的函数需要有一个参数<br>
此参数用来传数据,以字典形式传入<br>
可print来查看<br>
相关事件数据和序号参考
[JSR文档](http://game.xiafox.com/jsrdevdoc.htm#reg_t2)<br>
你也可以参考
[帮助](帮助.txt)<br>
返回值决定是否拦截（True继续，False拦截）<br>
